6、vuepress的使用

6、vuepress的使用

例如这张图片里的内容就是利用vuePress制作的

可以按照官方文档一步步做出来https://vuepress.vuejs.org/zh/guide/

在这里主要讲一下坑

  1. package.json里的是这样的
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs",
    "deploy": "bash deploy.sh"
  },
  1. 根目录下有个deploy.sh文件 ```bash

    !/usr/bin/env bash

    确保脚本抛出遇到的错误

    set -e

生成静态文件

npm run docs:build

进入生成的文件夹

cd docs/.vuepress/dist

如果是发布到自定义域名

echo 'www.example.com' > CNAME

git init git add -A git commit -m 'deploy'

如果发布到 https://.github.io

git push -f git@github.com:/.github.io.git master

如果发布到 https://.github.io/

git push -f git@github.com:websmallrabbit/testVuePress.git master:gh-pages

git@github.com:websmallrabbit/testVuePress.git 是发布到github上的仓库地址 gh-pages是分支名

cd -

```

  1. 我的vuePress项目的github地址是https://github.com/websmallrabbit/testVuePress
    下载安装源码
    1. git clone git@github.com:websmallrabbit/testVuePress.git
    2. cd testVuePress
    3. npm i
    4. yarn global add vuepress # 或者：npm install -g vuepress
    5. vuepress dev docs
